Class "Touch screen devices"
Attributes: 
name, brand, type of device, OS, CPU, size, 
color, screen(size, resolution, type of matrix) RAM,
camera (colors, main camera, front-camera), signal (ring tones, vibrating alert),
memory functions.
States: 
Turning on, turning off, old, new, previously used,
shabby, disassembled, stolen, infected by viruses, 
calling, vibrating.
Methods:
call, send message, make sound, 
connect to internet, storage data, 
vibrating, create photo, edit photos, delete photos,
upload applications, delete applications, 
find applications, change settings of device
